# Credit Risk Scorecard Monitoring System

## Overview
Bank-grade credit risk modeling project using:
- WoE Encoding
- Logistic Regression
- Scorecard Calibration
- PSI Monitoring
- Streamlit Dashboard

## Features
- Credit score generation
- Portfolio monitoring
- Drift detection
- Approval strategy simulator
- Risk segmentation

## Tech Stack
Python, Streamlit, Plotly, SQLite, Scikit-learn, Tableau
